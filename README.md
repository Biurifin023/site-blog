# Site Blog - Projeto Pratico

Este repositorio contem um projeto pratico desenvolvido durante uma aula de **Next.js** na plataforma **Rocketseat**.

O objetivo foi praticar os conceitos fundamentais do Next.js de forma simples e aplicada.

## Tecnologias utilizadas

- **Next.js 16** para estrutura da aplicacao e renderizacao.
- **React 19** para construcao da interface.
- **TypeScript** para tipagem e organizacao do codigo.
- **MDX** (`@next/mdx`, `@mdx-js/react`) para conteudo de posts.
- **Tailwind CSS** (+ PostCSS e Autoprefixer) para estilizacao.
- **ESLint** para padrao e qualidade de codigo.
